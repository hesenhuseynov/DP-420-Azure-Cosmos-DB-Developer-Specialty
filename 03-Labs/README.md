# 03 - Labs

This folder contains hands-on labs for practicing Azure Cosmos DB concepts with real examples.

The goal of this section is to turn theoretical knowledge into practical backend development experience.

## Purpose

Azure Cosmos DB concepts become much clearer when they are tested with real examples.

This section focuses on:

- Creating Azure Cosmos DB resources
- Working with databases, containers, and items
- Using the Azure Cosmos DB .NET SDK
- Understanding partition keys in practice
- Measuring RU consumption
- Testing indexing behavior
- Working with queries
- Using Change Feed
- Integrating Cosmos DB with Azure Functions and other Azure services

## Planned Labs

| No | Lab | Main Focus |

| 10 | Monitoring and Diagnostics | Logs, metrics, and troubleshooting |

| 12 | DevOps and Deployment Practice | Infrastructure and automation basics |
