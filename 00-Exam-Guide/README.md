# 00 - Exam Guide

This folder contains the exam strategy, official skill areas, study plan, and high-level roadmap for the DP-420: Azure Cosmos DB Developer Specialty certification.

## Purpose

Before going deep into Azure Cosmos DB topics, I want to clearly understand:

- What the exam measures
- Which topics are more important
- How the learning path is structured
- Which areas require hands-on practice
- Which topics are easy to confuse during the exam

## Files

| File | Purpose |
|---|---|

| Study-Plan.md | My study roadmap and weekly plan |
