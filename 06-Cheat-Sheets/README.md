# 06 - Cheat Sheets

This folder contains short, practical, and exam-focused cheat sheets for DP-420: Azure Cosmos DB Developer Specialty.

The purpose of this section is to quickly review important Cosmos DB concepts before the exam or before working on a real project.

## Purpose

Some Cosmos DB topics are repeated across many exam questions.

This section focuses on short and memorable summaries for:

- Partition key design
- Request Units
- Indexing strategy
- Consistency models
- Replication and failover
- Azure Cosmos DB .NET SDK
- Query optimization
- Change Feed
- Monitoring and troubleshooting
- Security and RBAC
- Backup and restore
- DevOps practices
